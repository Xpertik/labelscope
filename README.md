# labelscope

[![PyPI](https://img.shields.io/badge/pypi-unreleased-lightgrey.svg)](https://pypi.org/project/labelscope/)
[![License](https://img.shields.io/badge/license-Apache--2.0-blue.svg)](./LICENSE)
[![Python](https://img.shields.io/badge/python-3.10%2B-blue.svg)](https://www.python.org/)

Python SDK and viewer to preview thermal printer labels from their source code, without sending them to a physical device. Initial focus is **EPL2** (Zebra ZD410 and compatibles), with an architecture ready to grow into **ZPL** (and potentially TSPL, DPL).

See [`contexto.md`](./contexto.md) for the full spec, architecture, EPL2 command coverage, API design, and roadmap.

## Status

Early planning — no code yet. Scaffolding only.
